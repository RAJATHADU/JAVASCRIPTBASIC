JS-Variables

            DATATYPE
            1)Primitive Data Types
            2)Non-Primitive/Derived Data Types

            1)PRIMITIVE Data Types
                -Int(Integer)->It accepts only round numbers Ex=0-9
                -Float->It accepts decimal value Ex=10.7
                -Boolean->Either True or False
                -char->It accepts single character
                -void->returns null value

            2)Non-Primitive Data Types
                -Array    /*Collection of similar data types*/
                -Structure /* way to group several related variables into one place*/
                -Union /* allow to store different data types in same memory location*/
                -Enum    /*set of constant values*/Ex: all weekdays from monday to sunday,T-shirt sizes like S,M,L,XL etc..
                -Pointer /*variable that stores the address of another variable*/
                -Function  /*group of statements that together perform a task*/   
                -String /*Collection of characters.
